# jr_game_space
Игра реализована в псевдографике.  
Космический корабль (М) стреляет ракетами (R) в НЛО (U).  
А НЛО в свою очередь сбрасывает на корабль бомбы (B).
Корабль перемещаеться влево и вправо.
